npm init - команда для создания файла package.json

require - методя для вызова какого-то файла, который нам нужен

pathname - путь к файлу

app - зарезервированное слово, с допольнительными командами
app.get
app.listen

nodemon - пакет, который автоматически перезапускает сервер (устанавливать его нужно в dev,
по тому что после разработки этот пакет нам не нужен будет..)

npm run dev - команда запуска для пакета nodemon

body-parser - пакет, для анализа запроса пакета HTTP (например логин и пароль и т.д.)
ещё этот let bodyParser нужно require из ('body-parser')

app.post - команда в node.js, для создания, например, нового объекта...

postman chrome - приложения для имуляции того, что мы будем отправлять с нашего сайта на
сервер (устанавливается из google store) и получать обратно

.use - middleware function (промежуточная функция)

fetch(localhost:3012) -первый аргумент - путь
                       второй аргумент - header, body

request - запрос
response - ответ

API - application programming interface - набор готовых классов, процедур, функций и
структур - билблиотека;

REST - representation state transfer - архитектурный стиль взаимодействия компонентов;

rest full api;
    в протоколе http есть : get, post, put
    клиент сервер
    кеширование
    плагины

MVC - модель вид контроллер;
model - есть модель с логикой (api) - данные которые приходят от юзера для их обработки и
подальшей работы с серваком№;
view - за него отвечает react - то, что увидит наш юзер;
controller - следит за изменениями запросов пользователей на сервер - его основн. функция
- вызывать и координировать действие необходимых ресурсов и объектов, нужных для выполнения
действий, задаваемых пользователем;

OAuth - open authorisation - протокол авторизации, который позволяет предоставлять третьей
стороне ограниченный доступ к защищенным ресурсам пользоватьеля без необходимости
передавайть ей (третье стороне) логин и пароль


PassportJS - это middleware для авторизации под node.js. Он поддерживает авторизацию с
помощью огромного количества сервисвос, включая ВК и т.д.
passport -
passport-local - локальная стратегия, которую можно использовать для работы с mongoose
